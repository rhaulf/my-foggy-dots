# 🌲 Foggy Green Hyprland Rice

A minimalist, desaturated green and beige setup for Arch Linux.

## ✨ Components
- **WM:** [Hyprland](https://hyprland.org/)
- **Terminal:** [Ghostty](https://ghostty.org/)
- **Visualizer:** [Cava](https://github.com/karlstav/cava) (Foggy Green config)
- **Launcher:** [Rofi](https://github.com/davatorium/rofi) (Custom Foggy-Beige)
- **Icons:** Papirus-Dark (Nordic/Grey folders)
- **Cursor:** Bibata Modern Ice
